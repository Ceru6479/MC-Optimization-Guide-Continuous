### Celeritas

Celeritas is a port of Embeddium for various versions of Minecraft. This plans to be a list of all available versions with builds/jar-files. It is currently WIP.

You can find builds for various versions here: https://github.com/DHJComical/celeritas-mirror

Current known incompatibilities:

- Chunk Animator
- Embeddium (This is technically version 0.3.31)
- LittleTiles
- Nothirium
- Optifine
- Sodium (This is technically version 0.5.11)
- Vintagium & its forks

Shader support isn't currently available, however it is W.I.P.
