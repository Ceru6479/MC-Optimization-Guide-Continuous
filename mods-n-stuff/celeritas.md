### Celeritas

Celeritas is a port of Embeddium for various versions of Minecraft. This plans to be a list of all available versions with builds/jar-files.
Current version of the mod (as of 2/15/2026): 2.4.0-Dev3

Versions
Beta 1.7.3: Not available.
1.3.2: Not available.
1.4.7: Not available.
1.6.4: Not available.
1.7.10: Not available, use Angelica instead: [CurseForge](https://www.curseforge.com/minecraft/mc-mods/angelica) [Modrinth](https://modrinth.com/mod/angelica)
1.12.2: https://github.com/kappa-maintainer/Celeritas-auto-build/releases.
1.14.4: Not available.
1.16.5: Not available.
1.18.2: Not available.
1.19.2: Not available.
1.20.1: [Not available, use Embeddium instead.](https://www.curseforge.com/minecraft/mc-mods/embeddium)
1.21.1: Not available, use Embeddium or Sodium instead.

As of 2/14/26; Celeritas will run on Forge 1.12.2 without any further modification, but I can't guarantee mod stability.

Current known incompatibilities:

- Chunk Animator
- Embeddium (This is technically version 0.3.31)
- LittleTiles
- Nothirium
- Optifine
- Sodium (This is technically version 0.5.11)
- Vintagium & its forks

Shader support isn't currently available, however it is W.I.P.
